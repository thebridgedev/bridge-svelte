# Learning Documentation

This directory contains the Bridge Svelte learning docs in Markdown.

## Current structure

- **`learning/quickstart/quickstart.md`** — Quickstart guide: install, configure, and protect routes with Bridge.
- **`learning/examples/examples.md`** — Detailed examples: auth, feature flags, payments, configuration.

These are the main docs linked from the root README.

## Optional: MDX pipeline

If you use the MDX authoring workflow, the layout may also include:

- **`learning/mdx/`** — Authoring MDX files with `<Section>` wrappers
- **`learning/md/`** — Plain Markdown outputs (generated by the extractor)
- **`learning/scripts/`** — Utilities (e.g. MDX → MD extractor)

To extract plain `.md` from MDX (when available):

```bash
# Single combined .md
bun learning/scripts/extract-md-from-mdx.ts learning/mdx/quickstart.mdx learning/md/quickstart.md

# Combined + per-section files
mkdir -p learning/md/quickstart
bun learning/scripts/extract-md-from-mdx.ts learning/mdx/quickstart.mdx learning/md/quickstart/_all.md --split --outdir learning/md/quickstart
```

In that workflow, docs are written in `.mdx` with `<Section id="..." title="...">` wrappers, then exported to `.md` for GitLab or other destinations.
